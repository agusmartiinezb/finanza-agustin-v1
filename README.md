# Finanzas Agustín

App personal de control de gastos, ingresos e inversiones.

## Funcionalidades
- Registro de egresos, ingresos y transferencias entre cuentas
- Compras en cuotas que se proyectan automáticamente
- Multimoneda (ARS / USD)
- Presupuesto mensual por categoría
- Dashboard de seguimiento Plan vs Real
- Exportación a Excel
- Optimizada para celular

## Stack
React + Vite + Tailwind CSS

Los datos se guardan localmente en el navegador (localStorage).
